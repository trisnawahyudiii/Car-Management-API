### Built With

-   ExpressJS
-   Sequelize
-   Postgres
-   Swagger Open API

### Installation

1. Clone the repo
    ```sh
    git clone https://github.com/trisnawahyudiii/Car-Management-API.git
    ```
2. Install packages
    ```sh
    yarn install
    ```
3. Next Step

    ```sh
    - setting file database.js
    - yarn db:create; yarn db:migrate; yarn db:seed
    ```

4. Run

    ```sh
    yarn develop
    ```
